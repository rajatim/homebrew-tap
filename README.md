# Homebrew Tap

rajatim 的 Homebrew Tap，提供非官方套件。

## 安裝方式

```bash
brew tap rajatim/tap
brew install zhtw
```

## 可用套件

| Formula | 說明 |
|---------|------|
| [zhtw](Formula/zhtw.rb) | 簡轉繁台灣用語轉換器 - 讓你的程式碼說台灣話 |

## 更新套件

```bash
brew update
brew upgrade zhtw
```
